# ミニブログアプリ



## 概要
基本的なWebアプリケーション開発の理解を深めるため、
Laravelを用いたシンプルなブログ投稿アプリを作成しました。

前回制作したアプリは、入力内容に応じてAPIで所得したデータを表示するというシンプルな構成だったため、  
今回はデータベース操作やCRUD処理、認証機能など、
Laravelの主要機能を活用したアプリケーションとして制作しました。  
シンプルな投稿アプリをベースに、自分で機能の追加や改良をしながら制作しました。

## 開発環境・使用技術
*   XAMPP
*   Laravel 12.22.1
*   Laravel Breaze
*   bootstrap 
*   Database: MySQL


## 機能概要
*   ログイン機能 (Laravel Breaze)
*   記事CRUD機能 (投稿、編集、削除)  
            ◦コメント機能  
            ◦削除時ダイアログ表示(javascript)  
*   Gateによるユーザー認可
*   ペジネーション

## 工夫した点・技術的なアピールポイント
ログイン状態やユーザー権限に応じて、ページへのアクセス制御や画面表示を変更する処理を実装しました。  
Gateを利用した認可処理 を導入し、投稿者のみが記事を編集・削除できるようにしたり、  
未ログインで新規投稿をクリックすると、ログイン画面に遷移する仕組みになっています。  
また、UIは主にbootstrapを使用してシンプルに整えました。
